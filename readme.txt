

### ⚙️ 기술 스택 확정

| 구성    | 선택 기술              | 이유                       |
| ----- | ------------------ | ------------------------ |
| 프론트엔드 | React (Vite)       | 빠른 개발, 유연한 컴포넌트 구조       |
| 백엔드   | Node.js (Express)  | 경량, 내부망 호환성 우수           |
| DB    | SQLite             | 설치 간편, 파일 기반, 내부망 적합     |
| 인증    | 로그인 생략 또는 JWT (선택) | 초기 버전은 무계정 운영, 필요시 확장 가능 |

---

### 🧭 최종 메뉴 구성안 (Sidebar 기준)
📊 대시보드
   ├─ 오늘 할 일 요약
   ├─ KPI 달성률
   ├─ 공지사항 / 일정 알림
   └─ 휴가 중인 인원

📁 성과 관리 (KPI)
   ├─ 내 KPI 기록
   ├─ 팀별 목표 설정 (OKR)
   └─ KPI 달성 현황판 (그래프, 리포트)

✅ 실행 관리
   ├─ 나의 실행 계획 (To-do)
   ├─ 팀 단위 체크리스트
   └─ 상태별 보기 (예정 / 완료 / 중단)

📂 업무 게시판
   ├─ 자유 게시판
   ├─ 팀별 공유 게시판
   └─ 프로젝트별 회의록 / 자료실

📅 공용 캘린더
   ├─ 팀 일정 보기 (주간/월간)
   ├─ 배포 일정 관리
   ├─ 회의 / 외근 / 교육 일정
   ├─ 연차 / 반차 / 병가 신청 및 현황
   └─ 개인 KPI / TODO 마감일 연동

📬 공지 / 소통
   ├─ 공지사항
   ├─ 팀 공지 (역할별 가시성 설정)
   └─ 익명 제안함 (선택)

👥 사용자 / 권한 관리 (관리자)
   ├─ 사용자 목록 및 초대
   ├─ 역할 및 접근 권한 설정
   └─ 휴가 승인 / 계정 잠금

⚙️ 시스템 설정 (관리자)
   ├─ 업무/KPI 카테고리 설정
   ├─ 캘린더 색상/태그 설정
   ├─ 초기화 및 백업
   └─ 로그 기록 / 시스템 상태


---
✅ 구성 특징 요약
영역	핵심 목적
성과 중심 메뉴	KPI/OKR 기반 → 목표 설정부터 달성까지 연결
실행 추적 기능	개인/팀 단위 업무 진행률 추적 가능
공용 캘린더	휴가, 배포, 회의 등 통합 일정 관리 가능
게시판/소통	자유로운 업무 커뮤니케이션 & 공지 전달
관리자 기능	사용자 관리 + 일정 승인 + 백업 설정 등 실무 운영 대응




📄 1. 대시보드 페이지
/dashboard
오늘 할 일 요약 (To-do 요약)
KPI 진행률 그래프
공지사항 알림
오늘 휴가자 목록
캘린더 오늘 일정 하이라이트

📄 2. KPI / 성과 관리 페이지
/kpi/my – 나의 KPI 작성/편집
/kpi/team – 팀별 목표(OKR) 설정
/kpi/summary – KPI 달성 현황판 (차트/리포트)

📄 3. 실행 계획 (To-do) 페이지
/todo/my – 내 할 일 등록 및 상태 변경
/todo/team – 팀 단위 체크리스트 보기/관리
상태 필터 (예정/완료/중단) 탭 포함

📄 4. 업무 게시판 페이지
/board/free – 자유 게시판
/board/team – 팀별 공유 게시판 (팀별 필터)
/board/project – 프로젝트 회의록 / 자료 모음

📄 5. 공용 캘린더 페이지
/calendar
[탭 또는 필터]
전체 일정 보기
배포 일정
회의/교육
휴가 현황
KPI/To-do 연동 마감일

📄 6. 공지 및 소통 페이지
/notice/global – 전체 공지사항 목록
/notice/team – 팀 전용 공지사항
/suggestion – 익명 제안함 (선택 기능)

📄 7. 사용자 / 권한 관리 페이지 (관리자 전용)
/admin/users – 사용자 목록/등록/삭제
/admin/roles – 역할 권한 설정
/admin/vacations – 휴가 승인/관리

📄 8. 시스템 설정 페이지 (관리자 전용)
/admin/settings
KPI/업무 카테고리 관리
캘린더 태그/색상 설정
초기화, 백업, 로그 확인

✅ 선택적 추가 페이지
경로	용도
/login	JWT 로그인 (도입 시)
/error/403	접근 권한 없음 페이지
/error/404	페이지 없음 안내
/profile	마이페이지 (내 정보, 내 KPI/일정 등)
